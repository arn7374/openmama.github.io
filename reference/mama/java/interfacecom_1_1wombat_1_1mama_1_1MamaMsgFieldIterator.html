Detailed Description

Callback interface for MamaMsg.iterateFields().

An object implementing this interface is passed to the MamaMsg.iterateFields() method. The onField() method is invoked for each field in the MamaMsg object.

Method invoked for each field in the underlying message.

Please note that the same single instance of the MamaMsgField is passed to this method in all callbacks during iteration on a single message. The state of the object is set for each invocation. This decision helps to reduce unnecessary creation of many new, short lived, objects when parsing messages. As such is is recommended that references to MamaMsgField instances are not maintained after the completion of the onField method.
