native void com.wombat.mama.MamaQueue.destroy ( )

Destroy the queue, this function should only be called if there are no open objects against the queue.

Use one of the other destroy functions to block until all objects have been cleaned up.

Exceptions
MamaExceptionwill be thrown if there are open objects.

§ destroyWait()

native void com.wombat.mama.MamaQueue.destroyWait ( )

Destroy a queue.

Note that the queue can only be destroyed if all of the objects created on it, (timers, subscriptions etc), have been destroyed. This function will block until all of the objects have been destroyed and will then destroy the queue.
