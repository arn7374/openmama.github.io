§ create() [1/4]

void Wombat.MamaIo.create ( MamaQueue  queue,
MamaIoCallback  action,
uint  descriptor,
mamaIoType  ioType,
object  closure 
)

Create an IO handler.

If the underlying infrastructure does not support the requested mamaIoType, create throws MamaException(MAMA_STATUS_UNSUPPORTED_IO_TYPE). For example, RV only supports READ, WRITE, and EXCEPT. LBM supports all types except ERROR

Parameters
queueThe event queue for the io events. null specifies the Mama default queue
actionThe callback to be invoked when an event occurs.
descriptorWait for IO on this descriptor.
ioTypeWait for occurences of this type.
closureThe closure that is passed to the callback.
