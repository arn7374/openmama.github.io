§ mamaIo_create()

MAMAExpDLL mama_status mamaIo_create ( mamaIo *  result,
mamaQueue  queue,
uint32_t  descriptor,
mamaIoCb  action,
mamaIoType  ioType,
void *  closure 
)

Create a IO handler.

If the underlying infrastructure does not support the requested mamaIoType, mamaIo_create returns MAMA_STATUS_UNSUPPORTED_IO_TYPE. For example RV only supports READ, WRITE, and EXCEPT. LBM supports all types except ERROR.

Parameters
[in]resultA pointer to the io handle.
[in]queueThe event queue for the io events. NULL specifies the Mama default queue.
[in]actionThe callback to be invoked when an event occurs.
[in]descriptorWait for IO on this descriptor.
[in]ioTypeWait for occurrences of this type.
[in]closureThe closure that is passed to the callback.
Returns
mama_status return code can be one of: MAMA_STATUS_NULL_ARG MAMA_STATUS_INVALID_QUEUE MAMA_STATUS_NO_BRIDGE_IMPL MAMA_STATUS_OK
