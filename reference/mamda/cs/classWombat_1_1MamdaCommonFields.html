)
static

Set the dictionary for common update fields.

Maintains a cache of MamaFieldDescriptors for common update related fields. The properties parameter allows users of the API to map the common dictionary names to something else if they are beig published under different names.

Parameters
dictionaryA reference to a valid MamaDictionary
propertiesA NameValueCollection object containing field mappings. (See MamdaFields for more information)
