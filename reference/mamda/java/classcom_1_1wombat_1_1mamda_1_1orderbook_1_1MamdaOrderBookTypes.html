§ MAMDA_BOOK_REASON_MODIFY

final char com.wombat.mamda.orderbook.MamdaOrderBookTypes.MAMDA_BOOK_REASON_MODIFY = 'M'
static

Erroneous side (should not occur).

An enumeration for a reason for a change. Some of the values of Reason can mean the same thing, as far as their affect on the order book. If possible, a feed will send MODIFY, CANCEL or TRADE actions so that downstream applications that are interested in such data can handle it; other applications can treat such actions in the same way as an UPDATE action (or as a DELETE action if the size is zero).
